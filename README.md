## raxi
![npm](https://img.shields.io/npm/v/raxi) 

Toolkit for modern typescript packages.

- Mininaml
- Fully typed 
- Extendable 


## Getting started

```
pnpm dlx raxi init
```

Select the setup you want to use.

Want to use the latest commit?

clone then

```bash
git clone https://github.com/veritem/raxi.git
```

```bash
pnpm install
```

```bash
pnpm link --global
```
